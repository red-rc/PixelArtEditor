# Pixeller

**Pixeller** is a professional image editor focused on precision and control. It is designed for creating game textures, fonts, pixed-precise, technical and scientific images, and pixel art where accuracy matters.

---

## Features

- **Pixel-perfect editing**: Every pixel is fully controlled, with accurate zoom and grid support.
- **Frame-by-frame animation**: Create animated sprites directly in the editor.
- **Sprite sheet export**: Export animations as organized sprite sheets for game engines.
- **Layers and metadata**: Full support for layers, history, undo/redo, and image metadata.
- **Customizable interface**: Dockable panels, optimized workflow for projects.
- **Cross-platform**: Runs on Windows, Linux and macOS via Avalonia UI.

> Color management is planned for future updates, but current priority is on data and pixel accuracy.

---

## About

Pixeller was created to provide a professional-grade editor for artists who need precise control over every pixel. It is not a photo editor; it focuses on technical accuracy, animation, and game-ready assets.
